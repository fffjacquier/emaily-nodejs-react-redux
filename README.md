# A learning nodejs and react-redux project to manage survey campaigns via send grid and heroku

## Start the project locally

Start front and back by running:

```
npm run dev
```

## Back end uses

- nodejs
- mongoose
- express
- passport
- stripe
- localtunnel

## Front-end is in the client folder and uses

- axios
- materialize-css
- react
- react-router-dom
- redux, redux-form
- redux-thunk
